# 1.1. 用户状态监测

桌面客户端会一直进行用户状态检测和验证，有两种情况下会进行用户异常状态提示：
第一，当前登录信息失效，可能为登录过期等情况，会进行提示，需要退出到登录界面重新登录；
第二，当前网络异常，与后台的心跳一直处于异常状态，会进行调试，需要退出到登录界面重新登录。

# 1.2. 单端多云开发能力

作为CS端，除了可以开发本端外，还可以连接到多个云上进行开发，所以单端默认的应用（运维、审计、安全、时空功能开发、时空对象管理等）均支持多云开发模式。
交互如下：
![](media/单端多云开发.png)
在CS单端中，如果已存在相关账号登录，则直接打开，默认会显示统一登录管理中的第一个已登录的客户端，可以通过顶部栏进行删除和添加，点击添加后会弹出客户端选择弹出，在弹窗内选择添加，也可以直接跳转到统一登录中进行账号管理。
