# 2.2. 删除接口变量

**int32 EraseGlobal(string globalVariableName);**
**功能描述：**在APP运行期间手动删除全局变量，删除后的全局变量无法再通过GetGlobal()函数进行获取；APP对象删除时，该APP对象作用域内的所有全局接口变量会随之自动删除。
**注意事项：**
删除接口变量时，会取消关联订阅项、删除关联接口通讯链路；
删除服务类接口变量时，会断开和客户端的会话连接，断开对外的服务通讯；
删除客户端类接口变量时，会和服务断开连接，结束和服务的通讯；
如果全局变量名不存在，执行时会抛脚本异常；

string
globalVariableName

0：成功；
非0：失败，详见《07错误码_接口变量》；

 int32 rowLength = recv.rowLength;
 if(rowLength > 0)
 {
 // recv取第一行数据为例
 int32 eventType = recv.cellByName(1, "eventType"); // 接收事件类型, 详见系统枚举:Enum_IF_EventType
 blob data = recv.cellByName(1, "data"); // 接收数据
 Trace(Enum_LogLevel::DEBUG, "-D- [Serial]全局-订阅回调-接收-sucess-recv: ", ConvertToJsonString(recv, true));
 }
 else
 {
 Trace(Enum_LogLevel::DEBUG, "-D- [Serial]全局-订阅回调-接收-sucess-recv为空: ", ConvertToJsonString(recv, true));
 }
 }
 else
 {
 Trace(Enum_LogLevel::ERROR, "-E- [Serial]全局-订阅回调-接收-fail: ", errorCode);
 }
 Trace(Enum_LogLevel::DEBUG, "-D- [Serial]全局-订阅回调-接收-callback: ", ConvertToJsonString(callback, true));
}

{
 Serial serialClient = GetGlobal("serialClient");

 }
 else
 {
 Trace(Enum_LogLevel::ERROR, "-E- [Serial]全局-取消订阅-fail: ", errorCode);
 }
}
