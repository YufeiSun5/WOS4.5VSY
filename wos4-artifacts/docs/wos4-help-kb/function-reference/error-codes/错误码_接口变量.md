WOS_ERROR_CODE_ACCESS_DENIED
-70101
尝试绑定到特权端口（端口号小于1024）而没有管理员权限;尝试连接到受防火墙规则阻止的地址/端口
绑定到非特权端口(>= 1024); 配置防火墙允许所需连接;如果需要绑定特权端口以管理员/root 身份运行程序
WOS_ERROR_CODE_ADDRESS_FAMILY_NOT_SUPPORTED
-70102
尝试使用IPv4/IPv6地址族但系统或网络配置不支持
检测系统支持的地址族 (boost::asio::ip::tcp::v4(), boost::asio::ip::tcp::v6() 是否可用;确保操作系统启用了所需的网络协议栈
WOS_ERROR_CODE_ADDRESS_IN_USE
-70103
请求绑定的本地地址（IP + 端口）已被另一个套接字占用;套接字关闭后立即尝试重新绑定同一端口,而原套接字仍处于 TIME_WAIT 状态
如果原因是 TIME_WAIT，等待一段时间（通常几分钟）再重试;使用 SO_REUSEADDR 选项允许绑定到处于 TIME_WAIT 状态的地址;尝试绑定到另一个端口
WOS_ERROR_CODE_ALREADY_CONNECTED
-70104
尝试在一个已经成功连接的套接字上再次调用 connect()
避免重复连接
WOS_ERROR_CODE_ALREADY_STARTED
-70105
前一个连接操作还未完成，又尝试了新的连接操作
避免重复启动
WOS_ERROR_CODE_BROKEN_PIPE
-70106
向一个已经被对端关闭的套接字写入数据.对端应用程序关闭了它的套接字（正常关闭）;对端进程崩溃或被终止;网络中断导致连接断开
立即停止尝试向该套接字写入任何数据;调用 close() 或 shutdown() 清理本地资源
WOS_ERROR_CODE_CONNECTION_ABORTED
-70107
连接在建立过程中被本地软件（通常是操作系统或本地防火墙）中止
检查防火墙确认本地防火墙没有阻止出站连接;尝试重新连;接使用网络工具 (ping, traceroute, netstat) 检查基本连通性
WOS_ERROR_CODE_CONNECTION_REFUSED
-70108
目标服务器上没有运行监听指定端口的应用程序;目标服务器上的应用程序崩溃或未启动;目标主机上的防火墙规则阻止了入站连接到该端口;
检查目标主机地址和端口号是否正确;确认目标主机防火墙允许连接到该端口;确保目标服务器上的应用程序正在运行并监听目标端口（netstat -tuln）;重试
WOS_ERROR_CODE_CONNECTION_RESET
-70109
连接被对端重置。这通常是对端应用程序非正常关闭或网络问题.对端应用程序崩溃或异常终止;对端套接字在仍有数据待读的情况下被强制关闭;网络设备（如防火墙、NAT）超时并杀死了空闲连接;对端发送了TCPRST包（可能是协议错误或资源耗尽）
立即停止对该套接字的读写操作;服务端清理该连接的资源;客户端可以选择重新建立连接并重试
WOS_ERROR_CODE_CONNECTION_BAD_DESCRIPTOR
-70110
使用了无效的文件描述符。套接字已被关闭 (close());传递给函数的描述符值无效或从未打开过;
代码内部功能实现存在逻辑错误
WOS_ERROR_CODE_CONNECTION_FAULT
-70111
向套接字函数传递了无效的缓冲区指针(nullptr 或指向已释放内存)
代码内部功能实现存在逻辑错误，检查调用的ASIO-api传递的缓冲区是否正确
WOS_ERROR_CODE_HOST_UNREACHABLE
-70112
中间路由器返回了ICMP主机不可达消息。通常是由于路由问题或者网络物理断开或严重拥塞
确保目标 IP 地址正确;使用 ping 测试基本 IP 连通性;使用 route print (Windows) 或 ip route/netstat -rn (Linux/macOS) 检查路由表

WOS_ERROR_CODE_MESSAGE_SIZE
-70116
消息太大无法发送。UDP数据包大小超过了路径 MTU，并且套接字未设置 IP_DONTFRAG/IP_MTU_DISCOVER 选项允许分片(或者分片被防火墙阻止)

WOS_ERROR_CODE_NETWORK_DOWN
-70118
本地网络接口已关闭或禁用。网线被拔出（有线）;WiFi 被禁用或断开连接;网络接口被管理员禁用;
检查网络物理连接,确认网线未拔出、WiFi未关闭,网络未断开;检查接口状态:使用 ip link /ifconfig /netsh interface show interface 确认接口是否 UP

WOS_ERROR_CODE_NO_DESCRIPTORS
-70121
进程或系统级别的文件描述符资源耗尽.EMFILE: 当前进程打开的文件描述符数量已达到其软限制或硬限制 (ulimit -n);整个系统打开的文件描述符总数已达到系统级限制(sysctl fs.file-max)
