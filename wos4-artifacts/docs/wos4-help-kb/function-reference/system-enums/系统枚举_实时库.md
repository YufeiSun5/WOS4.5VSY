{

SPECIFY_SPACETIME=2, // 指定时空；
ALL_SPACETIME=3, // 整个数据区；
};
时空标识类型参数（spaceTimeIdentifierType)、时空标识参数（spaceTimeXXX,XXX=Fullnames/Aliases/GUIDs/IDs/Fullname/Alias/GUID/ID）在spaceTimeMode==2时有效，这时需要用户指定时空，其他spaceTimeMode无效；
时空范围参数spaceTimeScope在spaceTimeMode==1或2时有效，这时，最终访问哪些时空和参数spaceTimeScope参数有关系，详见枚举Enum_Rt_SpaceTimeScope，其他spaceTimeMode无效；
