属性名称

名称
name
string
用户对象的名称
字符为汉字、数字、字母（区分大小写）、下划线。首字符可以为数字，不能为下划线。不能包含特殊字符，有效长度64个字符。不可重复，不可为空（汉字算1个字符）用户名不允许被修改。

guid
string
用户创建时生成器的全局唯一ID

id

只读，系统产生
描述
description
string
用户对象的描述
选填，可改；最长128字符，可为任何字符
密码有效期
passWordValidity
int32
密码有效期，到期后密码失效，用户无法登入
选填，可改；以小时为单位设置密码有效期，参数范围应为1-2160小时（90天），数值不在1-2160之间，按90天处理
密码失效期
passWordFailure
dateTime
根据密码有效期和用户创建的时间计算得出的具体失效时间
只读，系统产生
登录有效期
loginValidity
int32
登入后允许使用的时间，超时后会话失效
选填，可改，参数范围应为1-720小时（30天），参数单位为小时，数值不在1-720之间，按30天处理
登录失效期
loginFailure
dateTime
根据登入有效期和用户登入时间计算得出的具体失效时间
用户不可填写查询该字段，系统产生
密码
passWord
string
用户密码，加密属性
读写，可查到密文密码密码：应包含至少三种不同的字符类型。且不能与用户名相同。1.大写字母（A-Z）2.小写字符（a-z）3.数字（0-9）4.特殊字符（!@#$%^&*,.;:’|+-_）创建用户时必填
用户状态
userStatus
int32
用户状态
读写，系统产生，从认证服务同步//1.启用态2.禁用态

logOnStatus
int32
用户在线状态
只读，系统产生，从认证服务同步//身份认证会话状态：1.在线态、2.离线态、3.锁定态
是否初始态
isInitialStatus
bool
是否初始态，初始态登入时需要修改密码
只读，系统产生，新创建的用户是初始态，重置密码后为初始态。需要修改密码后变为启用态。

activeTime
int32
用户活跃期，用于配置用户多长时间未操作，自动登出。
读写。前端取值范围10min-30day。后台折算为秒进行限制范围。默认为30min。

int32
