# 镜像包管理

 

# 1.上传

点击【上传】按钮，弹出上传弹窗，用户可以选择对应的镜像包，只允许格式为*.zip，每次只能上传单个文件。
选择文件之后，会显示当前镜像包的名称，此时点击【开始上传】按钮进行上传。
上传过程中用户可以查看上传详情，包括上传进度和状态。在上传过程中可以取消上传。

## 1.上传

 点击【上传】按钮，弹出上传弹窗，用户可以选择对应的镜像包，只允许格式为*.zip，每次只能上传单个文件。

 选择文件之后，会显示当前镜像包的名称，此时点击【开始上传】按钮进行上传。

 上传过程中用户可以查看上传详情，包括上传进度和状态。在上传过程中可以取消上传。

 

 

# 2.删除

用户可以对已经上传的补丁包进行删除。删除支持批量操作。点击操作栏的【删除】按钮或者列表上方的【批量删除】按钮，进行删除操作。
