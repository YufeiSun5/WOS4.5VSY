# 准入配置管理

云中心接入节点或者端区域有两种方式，其一是远程安装的形式；其二是在本地安装完之后，通过在云中心配置准入规则，然后根据准入规则生成准入密钥，在本地完成密钥的导入。第二种方式注册流程如下图：
graph LR
开始-->本地安装节点或者固定端-->在云中心进行准入配置-->生成准入密钥-->本地导入密钥-->完成注册

# 1.增加节点准入配置

点击【新建】按钮，打开准入配置添加弹窗，准入配置信息包含以下字段：
准入类型：云节点/固定端，默认云节点；

准入ID：节点准入ID，必填项；

业务网关IP：准入业务IP，必填项，不限制输入的字段，可以输入任意字段；

系统网关IP：准入管理IP，必填项，不限制输入的字段，可以输入任意字段。

# 2.删除节点准入配置

删除节点准入配置支持批量操作，需要注意的是，对于已经成功注册到云运维管理中心的节点或者固定端，删除节点准入信息时对应的节点或者固定端会受影响。

# 3.节点准入密钥生成

只有根据准入配置生成对应的密钥，然后再本地将其加载，才可完成（向云中心的）注册。
具体操作为点击【生成准入密钥】按钮，生成密钥信息，生成后自动下载在本地。
