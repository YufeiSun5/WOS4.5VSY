# 升级

 升级可针对wos服务或者Linux系统，WOS服务升级以节点粒度进行升级。

# 1.1升级

点击【升级】按钮，弹出升级弹窗，用户需要配置以下参数：
升级模式：WOS服务 / Linux系统；

升级类型：云 / 端；

仓库选择：升级的对象所处于哪个仓库；

升级对象选择：会根据【升级类型】和【仓库选择】这两个参数自动枚举出对应的列表。即升级类型为云时，升级对象为节点列表；升级类型为端时，升级对象为固定端列表；

镜像包选择：系统会根据【升级模式】和【升级类型】这两个参数自动枚举出对应的镜像包列表。镜像包信息以“名称 - 版本”格式展示。

![001](media/upgradeinstall001.png)
配置完成后，点击【确定】按钮，开始升级，需要注意的是升级过程中会重启所有服务。升级过程中，可以由用户取消升级。

# 1.2升级记录查看

用户可追溯升级记录，当打开升级弹窗时，可以点击【查看升级记录】按钮，弹出查看升级记录的弹窗，此时默认展示过去7天的数据。升级记录允许用户（根据时间范围）删除。
