# 1.5. 应用收藏

鼠标移入任务图标时，可以进行关闭应用和收藏应用，收藏应用后，应用将固定到任务栏，方便快速打开，如图：
![](media/收藏应用.png)
时空功能开发、时空对象管理始终固定在任务栏上，无需收藏。
④图标大小：桌面上的应用图标，默认以中等图标展示；其他展示模式包括大(96)、中(48)、小图标(32)切换；切换入口在桌面的系统设置，如图：
![image-20240905101315776](E:\00桌面系统\02应用图标管理\media\展示模式切换.png)
