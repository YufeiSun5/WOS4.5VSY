# 机架管理

 机架是计算机节点实际的机房部署位置，用户可以通过机架进行计算机的分类。

 机架管理包含新建、编辑以及删除。

 

# 1.新建

点击【新建】按钮，输入名称和ID，即可完成机架的新建。
![002](media/rack002.png)

## 1.新建

 点击【新建】按钮，输入名称和ID，即可完成机架的新建。

 

# 2.编辑

点击【编辑】按钮，仅允许修改机架名称。
![003](media/rack003.png)

## 2.编辑

 点击【编辑】按钮，仅允许修改机架名称。

 

# 3.删除

删除支持批量操作，用户可以通过点击操作栏中的【删除】按钮或者机架列表上方的【批量删除】按钮完成删除操作。
![004](media/rack004.png)

## 3.删除

 删除支持批量操作，用户可以通过点击操作栏中的【删除】按钮或者机架列表上方的【批量删除】按钮完成删除操作。
