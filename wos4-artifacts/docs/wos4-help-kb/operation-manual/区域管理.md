# 区域管理

云区域是独立的系统，区域间的数据区能够进行单向同步。区域间支持管理数据和业务数据的同步。区域向外提供接收数据同步通道，包括IP、端口、用户、口令。
用户通过传输网关传输数据，传输时需要指定目标区域信息（目标区域ID、目标云IP等），由业务管理人员在运维管理中心提前先配置好区域信息。

# 区域管理

 云区域是独立的系统，区域间的数据区能够进行单向同步。区域间支持管理数据和业务数据的同步。区域向外提供接收数据同步通道，包括IP、端口、用户、口令。

 用户通过传输网关传输数据，传输时需要指定目标区域信息（目标区域ID、目标云IP等），由业务管理人员在运维管理中心提前先配置好区域信息。

 

# 1.1新建

新建区域时包含基本信息，包括名称、ID、类型；同时用户可以定义该区域的地址信息，针对业务传输、管理传输用途可配置具体的参数。
点击【新建区域】按钮，弹出新建区域弹窗，区域信息包含基本信息和连接信息。
新建完区域后，会在列表中添加一行，此时可以通过点击“名称”列的展开按钮来查看次级信息（包含业务传输网关、管理传输网关）。
![001](media/area001.png)

### 1.1新建

 新建区域时包含基本信息，包括名称、ID、类型；同时用户可以定义该区域的地址信息，针对业务传输、管理传输用途可配置具体的参数。

 点击【新建区域】按钮，弹出新建区域弹窗，区域信息包含基本信息和连接信息。

 新建完区域后，会在列表中添加一行，此时可以通过点击“名称”列的展开按钮来查看次级信息（包含业务传输网关、管理传输网关）。

 

# 1.2修改

修改区域时，ID参数不允许修改，其余参数可提供给用户修改。

# 1.3删除

支持删除多余或无效的云端区域。删除支持批量操作。
