# 证书管理

 安全客户端中负责证书的上传和启用，证书包含两种类型：http协议和TCP协议。当部署了https证书后，能够对客户端连接并访问云区域之间的通信进行加密，防止数据传输过程中被窃听、篡改或者伪造。当部署了TCP协议证书，客户端访问云区域或者不同云区域之间的传输可靠性会比较高。

 证书支持两种协议，分别是HTTP协议和TCP协议。

# 1.上传

点击【上传】按钮，选择具体的证书类型，进行证书的上传。证书类型需要为*.zip格式。

## 1.上传

 点击【上传】按钮，选择具体的证书类型，进行证书的上传。证书类型需要为*.zip格式。

 

 上传之后，可以在列表中看到证书的基本信息。

 

# 2.启用/禁用

同一类型的证书，同一时间只能启用一个，当启用一个的时候，系统会自动禁用其他已启用的证书。
点击操作栏栏中【启用】按钮，当状态变为“禁用”时，代表启用成功。
