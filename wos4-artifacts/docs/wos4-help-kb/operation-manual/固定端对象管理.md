# 固定端对象管理

 固定端一般为盒子或生产现场的小型数据采集设备，也称“微云”。固定端对象接入有两种方式：

 
 通过云中心远程安装的方式；

 本地安装后，在云中心注册。

 

 

# 1.新建&安装固定端

点击【新建】按钮，进行固定端的新建和安装。
配置信息包含：基本信息、连接信息、登录信息、报警阈配置、安装信息。
![001](media/endinstall001.png)
磁盘限制：300MB-10T；
![002](media/endinstall002.png)

## 1.新建&安装固定端

 点击【新建】按钮，进行固定端的新建和安装。

# 2.更新配置项

固定端在安装之后可以修改以下信息：名称。
点击操作栏中的【编辑】按钮，进行基本信息修改。
![](media/endinstall012-新增.png)
点击操作栏中的【报警策略配置】按钮，进行报警开关和报警阈的修改。

## 2.更新配置项

 固定端在安装之后可以修改以下信息：名称。

 点击操作栏中的【编辑】按钮，进行基本信息修改。

 

 点击操作栏中的【报警策略配置】按钮，进行报警开关和报警阈的修改。

 

# 3.启停端库进程

云中心可对接入的固定端，远程控制端库进程的启停。启停支持批量操作。点击操作栏【启动/停止进程】按钮或者列表上方【批量启动/停止】按钮，打开启停弹窗。
![006](media/endinstall006.png)
![007](media/endinstall007.png)

## 3.启停端库进程

 云中心可对接入的固定端，远程控制端库进程的启停。启停支持批量操作。点击操作栏【启动/停止进程】按钮或者列表上方【批量启动/停止】按钮，打开启停弹窗。

 

 

# 4.更新登入/登出状态（暂不支持）

可以在云中心远程更新固定端的登入和登出状态，更新时需要填写固定端连接云的功能单元用户名和密码。

# 5.查看硬件信息

固定端接入完成之后，运维服务通过端库上相应进程，能够获取到固定端的各类硬件信息，以下是硬件信息具体详情。包括CPU、内存、网卡、硬盘以及文件系统。

# 6.性能监视

固定端同节点一致，可以进行实时性能监视和历史性能记录查询，包括CPU、内存、磁盘以及网卡。
![009](media/endinstall009.png)

# 7.删除&卸载固定端

删除支持批量操作。卸载时因需要重新校验机器信息，故用户需要重新输入连接IP、用户名、密码和系统类型。点击操作栏【删除】按钮或者列表上方【批量删除】按钮，打开卸载详情弹窗。
![010](media/endinstall010.png)
