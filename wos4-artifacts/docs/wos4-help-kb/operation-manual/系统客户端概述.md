# 系统客户端概述

# 1.产品介绍

系统客户端可以对整个WOS系统进行配置，并管理WOS所有系统对象，包括区域、计算机节点、资源组、进程等的配置操作。系统管理中心能够监视系统对象的性能，并进行报警推送。
系统客户端作为云客户端，只有登录单端后，才可以进行访问，此时通过具体的云连接信息来访问。具体的操作步骤如下：
（1）点击【管理工具】图标按钮，输入具体的连接IP和标题、用户名和密码，点击【登录】按钮。
注意：这里需要**提前安装云环境**，才能进行具体的访问。

## 2.术语介绍

# 2.1区域

WOS云区域之间能构成多层次企业系统，如云中心 + 边缘计算 + 采集终端的三层体系，集团 + 工厂的二层体系。也能作为单机系统提供用户学习、开发、调试终端。
云中心由一组计算机节点构成（最少一个节点），云中心支持集群架构能够扩展计算机节点。多个设备端和云中心建立数据同步，多个客户端能够登录到云中心远程访问。

### 2.1区域

 WOS云区域之间能构成多层次企业系统，如云中心 + 边缘计算 + 采集终端的三层体系，集团 + 工厂的二层体系。也能作为单机系统提供用户学习、开发、调试终端。

 云中心由一组计算机节点构成（最少一个节点），云中心支持集群架构能够扩展计算机节点。多个设备端和云中心建立数据同步，多个客户端能够登录到云中心远程访问。

# 2.2机架

定义区域的机架信息。机架是机房安装计算机阶段的物理机架，计算机节点属于物理机架。

### 2.2机架

 定义区域的机架信息。机架是机房安装计算机阶段的物理机架，计算机节点属于物理机架。

# 2.3资源组

WOS云中心的系统服务采用集群架构，支持多实例多副本。集群由多个计算机节点构成（1~n），如果集群负载高时，可以扩容集群的节点。计算机节点多时每个集群是一组独立的节点资源，如果节点少时四类集群共用一组节点资源。多个集群之间节点资源交叉会导致调度效率下降。
集群是指计算机节点资源组，多个副本调度的节点范围。可以在系统客户端中对以下四个资源组进行管理：管理资源组、时空库资源组、功能单元资源组、网关资源组。
