# `wos4-artifacts/reports`

报告目录。

用于保存阶段报告、验证报告、人工复核记录、时间报告和可提交的分析结论。

报告应说明数据来源、验证方法、结论、未解决风险和关联证据路径。

## 和 tests 的区别

- `wos4-artifacts/tests/` 放测试用例、测试脚本、测试数据和可复跑步骤。
- `wos4-artifacts/reports/` 放测试执行后的汇总结论、阶段复盘、验证报告和给人看的决策材料。

## 报告类型

- 阶段报告：说明某一阶段做了什么、完成了什么、还剩什么风险。
- 验证报告：说明某次测试的输入、环境、步骤、结果和证据。
- 回归报告：说明历史问题是否复现、是否通过。
- 人工复核记录：说明人工观察到的 UI、截图和判断。
- 时间报告：记录耗时、执行窗口、失败重试和阻塞时间。

报告可以引用 `tests/`、`screenshots/`、`snapshots/`、`backups/` 中的证据，但不要复制大量原始数据。
