## 9. 代码插入组件方法

本页已验证的代码插入方法：

1. 先从 `rowsManager` 读取命名槽位
2. 再通过 `KSGUID` 找到真实 `kids.RCol*` 实例
3. 从 `.toolkit_group_item` 的 Vue 对象上取 toolkit
4. 深拷贝 toolkit
5. `targetCol.addChild(null, [clone], "drag")`
6. 通过 `targetCol.getObject()` 取回实例
7. 设置 `UIName / propData / detailConfig`
8. 提交保存

关键提醒：

- 不要直接对 `rowsManager` 里的列数据对象调 `addChild`
- browser-harness 的 ASCII 临时脚本里，中文要写成 Unicode 转义
- 图表和表格插入后要立刻补 `detailConfig`
