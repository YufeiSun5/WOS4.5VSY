# 03 前端逻辑

## 页面变量

| 变量名 | 类型 | 默认值 | 用途 |
|---|---|---|---|
| excelCellAddress | string | B1 | 当前实验单元格地址 |
| excelCellValue | string |  | 当前写入值 |
| excelLastResult | string |  | 最后一次操作结果 |
| excelCapabilityJson | string |  | Excel 组件探针摘要 |

变量是否实际创建，需以页面变量探针和组件事件持久化能力为准。

## 组件清单

| 组件 | WOS4 类型 | 所在布局槽位 | 关键配置 |
|---|---|---|---|
| 已有 Excel 组件 | 待探针确认 | ExcelHostCol | 保留原组件并读取 API |
| 单元格地址输入框 | ElementInput | CellAddressRow | 默认 `B1` |
| 写入值输入框 | ElementInput | CellValueRow | 支持文本/数字输入 |
| 操作按钮组 | ElementButton | OperationButtonsRow | 读取、写入、清空、样例填充 |
| 样式按钮组 | ElementButton | StyleButtonsRow | 背景、字体、边框、合并等按能力启用 |
| 导出/打印按钮组 | ElementButton | ExportPrintRow | 导出、打印、分页探针 |
| 结果日志 | 待定文本/表格 | ResultLogRow | 显示返回值、异常、边界结论 |

## 探针计划

先读取现有 Excel 组件：

- `UIName`、`prototypeName`、`toolkit id`、`KSGUID/KSCGUID`。
- `propData/detailConfig/styleConfig/linkList`。
- 公开方法名、方法参数数量、是否存在 `get/set/cell/range/export/print/page/header/footer/style` 相关入口。
- DOM 中是否有 canvas、table、iframe、luckysheet、spreadsheet、x-spreadsheet、handsontable 等特征。

## 交互流程

- 读取：地址输入框值 -> 调用 Excel 组件读取 API -> 写入结果日志。
- 写入：地址和值输入框 -> 调用写入 API -> 再读回校验。
- 样式：地址输入框 -> 设置样式 -> 截图/DOM/导出结果验证。
- 导出：触发组件导出或浏览器下载 -> 保存下载证据和文件摘要。
- 打印分页：构造多页数据 -> 设置或探测表头表尾 -> 触发打印/导出 -> 记录分页结果。

## 持久化要求

运行态探针只能作为能力发现；最终按钮脚本必须写入组件自身 `linkList` 或页面创建脚本等已验证持久化入口，并通过提交、重开、预览验证。
