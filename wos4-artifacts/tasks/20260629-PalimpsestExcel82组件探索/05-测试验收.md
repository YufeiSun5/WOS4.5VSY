# 05 测试验收

## 测试策略

- 集成测试：WOS4 页面保存、提交、重开、预览和按钮交互。
- 人工测试：导出文件、打印预览、分页表头表尾保真。
- 回归测试：确认只影响 `PalimpsestExcel_82`，不改其他 Palimpsest 页面。

## 用例清单

| 类型 | 用例 | 证据路径 | 通过标准 |
|---|---|---|---|
| 集成 | 打开指定页面并完成 before 备份 | `backups/`、`snapshots/` | 备份含 `before-runtime.json`、`restore-runtime.js`、截图 |
| 集成 | Excel 组件能力探针 | `snapshots/` | 记录原型、方法、属性、DOM 特征 |
| 集成 | 读取 B1 | `snapshots/`、预览截图 | 点击后日志显示 B1 值或明确不支持原因 |
| 集成 | 写入单元格并读回 | `snapshots/`、预览截图 | 写入后再次读取得到同一值 |
| 集成 | 设置样式 | `screenshots/` | 目标单元格视觉变化或明确不支持原因 |
| 人工 | 导出文件 | `snapshots/`、下载文件摘要 | 导出成功且记录内容/样式保真情况 |
| 人工 | 打印分页表头表尾 | `screenshots/`、报告 | 多页时记录表头/表尾是否自动分页 |
| 回归 | 重开编辑页 | `snapshots/` | 实验区布局和脚本仍在 |
| 回归 | fresh 预览页 | `screenshots/` | 不使用旧预览 tab，功能可点 |
