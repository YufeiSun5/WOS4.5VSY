# 测试验收

## 本轮测试

- 打开新建页面编辑器，确认画面渲染。
- 编辑器 runtime 中点击第一行行首 `展开`。
- 验证第一列从 `展开` 变为 `收起`，主表体行数从 6 变为 7，明细行数量从 0 变为 1。
- 验证明细内容包含部件明细标题、设备编码、健康度和维护建议。
- 验证行首没有 `?` 或异常图标，当前仅显示 `展开/收起` 文本。
- 验证操作列没有使用 fixed right，避免参考页同类错位。
- 验证视觉增强元素存在：`opButtons=18`、`statusPills=6`、`healthBars=6`。
- 验证左侧树节点存在：`treeNodes=8`。
- 验证点击 `二车间` 后右侧表格变为 2 条，且数据为二车间设备。
- 验证点击 `异常` 后右侧表格变为 1 条，且展开后显示 `仓储堆垛机` 明细。
- 验证默认查询结果第一条自动展开：`details=1`、`firstText=收起`、`firstOpen=true`。
- 验证所有查询和按钮都是前端假数据逻辑，不接后端。
- 保存截图和运行态快照。

## 暂不测试

- 后端增删改查。
- 蓝色客户端发布。
- 真实时空对象数据查询。
- 顶部提交版本和预览链路：当前工具栏按钮未能稳定 DOM 定位，本轮只完成编辑器 runtime 保存和点击验证。
