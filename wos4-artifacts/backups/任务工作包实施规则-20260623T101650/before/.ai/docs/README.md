# `.ai/docs`

项目设计和规则文档目录。

用于保存目录地图、备份策略、版本树、执行路线、清理计划和阶段性设计说明。

常用文件：

- `project-directory-map.md`：目录怎么编排。
- `backup-and-version-policy.md`：修改前备份和版本管理规则。
- `mcp-and-local-tools.md`：MCP、本地浏览器工具和用户侧安装/启用要求。
- `ai-interaction-panel-policy.md`：AI 交互面板 open/closed、关闭权限和归档规则。
- `wos4-version-tree.md`：WOS4 平台逻辑版本树。
- `wos4-clean-restart-runbook.md`：clean restart 主线执行说明。

具体自动化步骤应沉淀到 `.ai/skills/`，不要只停留在 docs。
