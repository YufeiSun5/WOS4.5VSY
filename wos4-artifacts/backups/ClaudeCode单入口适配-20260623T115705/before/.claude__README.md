# Claude Code 适配层

`.claude/` 只用于兼容 Claude Code。

项目真实 AI 规则仍在：

- `AGENTS.md`
- `.ai/agents/`
- `.ai/instructions/`
- `.ai/skills/`
- `.ai/docs/`

## 使用方式

在仓库根目录运行 Claude Code 时，`CLAUDE.md` 会导入 `AGENTS.md`。

如果需要显式调用项目 skill，可以使用 `.claude/skills/` 下的同名 wrapper，例如：

```text
/wos4-login
/wos4-layout-devtools-skill
/wos4-component-persistence
```

这些 wrapper 不保存真实流程，只要求 Claude Code 回读 `.ai/skills/<skill>/SKILL.md`。

## 维护规则

- 不要把 `.ai/skills/*/SKILL.md` 的正文复制到 `.claude/skills/`。
- 新增项目 skill 时，优先在 `.ai/skills/` 新增真实 skill；如需 Claude Code 直接调用，再增加同名 wrapper。
- 规则冲突时，以 `AGENTS.md` 和 `.ai/` 为准。
