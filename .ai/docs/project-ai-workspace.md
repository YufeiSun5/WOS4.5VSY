# 项目 AI 工作区说明

## 根目录入口

- `AGENTS.md`：AI 路引和强制规则。
- `MEMORY.md`：项目长期记忆和 AI 修改记录。
- `AI-INTERACTION-PANEL.md`：单文件 AI 协作看板。

## `.ai` 目录

- `agents/`：AI 身份定义。
- `instructions/`：全局和专项规则。
- `skills/`：可迁移 skill 索引和副本。
- `docs/`：方案文档和研究结论。
- `closed-interactions/`：已关闭交互归档。

## `wos4-artifacts` 目录

集中放置 WOS4 自动化过程中的脚本、截图、快照、测试、报告和备份。旧文件暂不移动，先通过 Git 建立基线。
