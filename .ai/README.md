# `.ai`

项目 AI 工作区。

本目录放 AI 身份、项目规则、项目 skill、设计文档和已关闭协作记录。所有 AI 开始任务前先读根目录 `AGENTS.md`，再按任务进入本目录读取对应文件。

## 子目录

- `agents/`：AI 身份定义。
- `instructions/`：项目级通用规则。
- `skills/`：项目真实 skill 源。
- `docs/`：项目设计、目录、备份、版本树和执行路线文档。
- `closed-interactions/`：已关闭协作事项归档。

不要在本目录放真实账号、密码、Cookie、Token 或浏览器 profile。
