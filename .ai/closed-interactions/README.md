# 已关闭交互归档

本目录保存从 `AI-INTERACTION-PANEL.md` 关闭并移出的历史事项。

## 强制规则

- 选择 close 的同一次操作中，必须完成归档、从当前面板移出、更新 `INDEX.md` 和更新 `MEMORY.md`。
- 不允许等 `AI-INTERACTION-PANEL.md` 变长后再集中归档。
- 归档文件必须写明 `状态：closed`、关闭触发、关闭人、关闭时间、关闭原因和最终结论。
- 归档文件名建议使用 `<日期>-<事项编号>-<简短中文标题>.md`。

## 路引文件

- `AI-ROUTE.md`：AI 查询历史事项时先读这里。
- `INDEX.md`：归档索引，按编号、标题、任务工作包、参与人和关键词定位具体文件。

查询历史事项时不要只按日期猜文件；先查 `INDEX.md`。
